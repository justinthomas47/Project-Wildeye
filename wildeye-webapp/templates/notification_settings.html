Notification Setup Guide

Setting Up Email Notifications

To receive email notifications:

  1. Enter a valid email address in the Email Notifications section
  2. Make sure the toggle is enabled
  3. Email alerts will be sent whenever an animal is detected

Setting Up SMS Notifications

To receive SMS notifications:

  1. Enter your mobile number with country code (e.g., +91XXXXXXXXXX)
  2. Make sure the toggle is enabled
  3. You can also set camera-specific phone numbers when adding cameras
  4. SMS alerts will be sent whenever an animal is detected

Setting Up Telegram Notifications

To receive Telegram notifications:

  1. Find the WildEye Telegram bot on Telegram
  2. Start a chat with the bot by sending "/start"
  3. Get your Chat ID by messaging @userinfobot on Telegram
  4. Enter your Chat ID in the Telegram Notifications section
  5. Make sure the toggle is enabled
  6. Telegram alerts will be sent whenever an animal is detected
